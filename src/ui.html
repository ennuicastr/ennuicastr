Help

Welcome to Ennuicastr.

Hopefully, your microphone was picked up automatically and is of good quality, in which case you're done! All you need to do is talk. You can use the waveform at the bottom of the screen to verify that audio capture is working. If necessary, you can mute yourself at any time by using the mute button ().

To invite guests, use the invite button at the top of the screen () and then copy the link. To start recording, use the “start recording” button in the quick menu at the top of the screen, or the same button in the host interface, which you can access with the host interface button () from the main menu. Remember, nothing is being recorded until and unless you start recording!

Use the “share camera” () or “share screen” () buttons to share your camera or screen video, respectively.

If the wrong input device is used or you have any other problems, use the settings screen () to change your settings. You can do things like decrease sensitivity to background noise, change your input device, change your camera, etc.

As the host, you can also change many settings for your guests, by using the user administration screen (). Sensitive settings will require permission from the guest, which you can also request from that screen.

You can use text chat () to communicate if you're having audio issues. In addition, text chat is recorded and timestamped, so it can be used to take show notes.

If you don't like the screen arrangement, several other arrangements are available in settings (). In particular, power users may want the “studio” view mode, which shows a waveform for all users.

If you like keyboard hotkeys, a hotkey can be assigned to any button in the user interface by control-clicking the button.

Close this dialog with the close button at the top-right (), the escape key on your keyboard, or by simply clicking outside of the dialog.
